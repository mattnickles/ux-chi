## 2.0.0 (April 23, 2020)

### Components
#### Breaking changes
* Renamed: All component class prefixes for consistency with Chi's web component names. (e.g. `a-icon` is now `lux-icon`).

* Renamed: Text utility size classes for consistency (e.g. `-text--smaller` is now `-text--xs`).
